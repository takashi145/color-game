# カラーゲーム - ストループ効果ゲーム 仕様書

## 1. ゲーム概要

ストループ効果を活用した脳トレゲームアプリ。画面に表示された色の名前（テキスト）について、文字色または文字の内容のどちらかを正確に答えることで、認知能力を鍛えるゲームです。

### ストループ効果とは
文字の意味（書かれている色の名前）と文字の色が異なる場合、どちらか一方に素早く反応することが困難になる心理現象です。例えば「赤」という文字が青色で表示されている場合、文字色を答えるのに時間がかかります。

## 2. ゲームモード

### 2.1 モード1: 文字色モード
- 表示された文字の**色**を答える
- 例: 「赤」という文字が青色で表示 → 正解は「青」

### 2.2 モード2: 文字内容モード
- 表示された**文字の内容**を答える
- 例: 「赤」という文字が青色で表示 → 正解は「赤」

### 2.3 モード3: ミックスモード（上級）
- ランダムに「文字色を答えろ」「文字内容を答えろ」が切り替わる

## 3. 使用する色

基本的な色セット（6色）:
- 赤 (Red)
- 青 (Blue)
- 緑 (Green)
- 黄 (Yellow)
- 紫 (Purple)
- オレンジ (Orange)

## 4. ゲームフロー

### 4.1 タイトル画面
- ゲームタイトル表示
- モード選択ボタン
  - 文字色モード
  - 文字内容モード
  - ミックスモード

### 4.2 ゲーム画面

#### 表示要素
- 上部ヘッダー
  - 現在のスコア
  - 残り時間（カウントダウン）
  - 制限時間のバー表示
- 中央エリア
  - モード指示（ミックスモードの場合）
  - 色の名前（テキスト）を大きく表示
  - テキストの色と内容は異なる組み合わせ
  - 正解/不正解のフィードバック（アニメーションや色のフラッシュ）
- 下部エリア
  - 回答ボタン（6色分）
  - 各ボタンには色の名前と対応する背景色

### 4.4 ゲームルール

#### 時間制限
- 60秒以内にできるだけ多くの問題に回答(秒数は調整可能にするかも)
- 各問題は3秒以内に回答

### 4.5 結果画面
- 最終スコア表示
- 正解数/総問題数
- 正答率
- ハイスコア更新の場合、お祝いアニメーション
- リトライボタン
- ホームに戻るボタン

## 5. UI/UX仕様

### 5.1 カラースキーム
- 背景: 白またはライトグレー
- テキスト: 視認性の高いフォント
- ボタン: それぞれの色を背景色に、テキストは黒または白（視認性重視）

### 5.2 フォント
- 問題テキスト: 太字、大きめ（48-64px相当）
- ボタンテキスト: 中サイズ、読みやすい（18-24px相当）
- スコア/時間: 明確に見える（20-28px相当）

### 5.3 アニメーション
- 正解時: 緑のフラッシュ + ○マーク
- 不正解時: 赤のフラッシュ + ×マーク

## 6. データ仕様

### 6.1 問題生成ロジック
```
問題 = {
  テキスト内容: ランダムな色名,
  テキスト色: テキスト内容とは異なるランダムな色,
  正解: モードに応じた答え
}
```

### 6.2 保存データ
- ハイスコア（モード別）
- 総プレイ回数
- 総正解数/総問題数
- 設定（サウンドON/OFF等）

## 7. 技術仕様

### 7.1 プラットフォーム
- Flutter（iOS/Android対応）

### 7.2 状態管理
- StatefulWidget または Provider/Riverpod

### 7.3 データ永続化
- shared_preferences パッケージを使用

### 7.4 必要なパッケージ
```yaml
dependencies:
  flutter:
    sdk: flutter
  shared_preferences: ^2.0.0  # データ保存
  # オプション:
  # audioplayers: ^x.x.x  # サウンド
  # vibration: ^x.x.x  # バイブレーション
```
